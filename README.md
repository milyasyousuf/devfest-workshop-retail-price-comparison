# Gemini AI Retail Price Comparison

A Python Program that uses Google's Gemini AI to analyze PDF product specifications and compare prices.


## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Set up API key:**
   ```bash
   echo "GEMINI_API_KEY=your_api_key_here" > .env
   ```

3. **Add PDF spec sheets to `data/` folder**

4. **Run analysis:**
   ```bash
   python program.py
   ```

## What It Does

- Extracts product specs from PDF files using Gemini AI
- Creates price comparison tables
- Provides AI-powered recommendations
- Saves results to `demo_results.json`

## Requirements

- Python 3.8+
- Google Gemini API key
- PDF specification sheets

## Features

- Multimodal PDF analysis
- Automated data extraction (processor, RAM, storage, price, battery, weight)
- Price comparison tables
- AI-powered best value analysis
- JSON output for integration
